# Analyse statistique – EL AMAL HUILERIE

##  Contexte
Ce projet a été réalisé dans le cadre de l’évaluation du module d’analyse de données.
Il porte sur l’étude statistique des données de production et de prix de la société
EL AMAL HUILERIE.

##  Objectifs
- Nettoyer et préparer les données brutes
- Réaliser une analyse statistique descriptive
- Visualiser les distributions et tendances
- Répondre à des questions d’analyse statistique

##  Données
- Données brutes : Excel
- Données nettoyées : CSV
- Variables principales : prix_total, quantités, dates

##  Outils utilisés
- Google Colab (R)
- tidyverse, ggplot2, lubridate, janitor
- Quarto
- GitHub

## Méthodologie
1. Importation des données
2. Nettoyage (types, dates, doublons)
3. Analyse statistique descriptive
4. Visualisations graphiques
5. Tests statistiques (Shapiro, t-test, régression, corrélation)

##  Résultats
- Distribution du prix total
- Évolution temporelle des prix
- Relation entre quantité et prix

##  Contenu du dépôt
- `data/` : données brutes et nettoyées
- `notebooks/` : notebook Google Colab
- `quarto/` : présentation Quarto
- `slides/` : support de présentation
